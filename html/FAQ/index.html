Questions about the server (now out of date)

What hosting service do you use?
Pfft... Hosting! Ha!
The server is a Raspberry Pi running Apache on Raspbian Linux. I usually only have a power and ethernet cable attached to it, and use PuTTY to SSH onto it and develop the website from there. I also use WinSCP I keep the server behind my TV cabnet when it is running, because that's the only way the cables reach. When running, the Pi stays in a Lego case that keeps it cool and protects it from blunt force trauma.
What do you use to make the webpages?
I use GNU Nano and the WinSCP default editor, mostly. Since the Pi is, well, just a Pi, I upload all images to imgur, and just reference them from there. (My imgur account is 1421616168.imgur.com)
