Inside the Black Box

Most people consider a computer to be a black box, they have a mental image of what a computer is, but no idea what is inside it. They think of the computer as one object, one mental group, rather than the crazy cumulation of the past couple decades of ideas. My goal with this is very similar to that of The Joy of X1, in that I would like to start with the very basics of how a computer works, and work up to the level that they are at now. We'll start with the very basics of boolean logic and work our way up to the operating system level.

Boolean Logic

In the math that you are used to, there are an infinite amount of different numbers to use; they just don't end. In boolean math, or logic, however, there are exactly two different numbers which correspond to true and false. These are the infamous 'ones and zeros' that are 'inside' computers. If you've ever believed anyone telling you about straightening cables so that the ones and zeros don't get stuck, you may want to quit now. Anyways, instead of the normal mathematical operations + - * /, there are only three boolean operations: and, or, and not. These are pretty self explanatory in that they work pretty much exactly the same as in english. The only notable difference is the or operator. If someone asks for one thing or another, having both things is not part of what they meant. We can use what are called 'truth tables' to help clarify the meaning of these operators. A truth table looks like this:

Left Right Result
false false false
true false true
false true true
true true true

This is, in fact, a truth table for the or operator: Left and Right are the left and right operands of the or operator, respectively, and Result is, of course, the result of the operation. We can define truth tables for all three of the operators as follows:

and:

Left Right Result
false false false
true false false
false true false
true true true

or:

Left Right Result
false false false
true false true
false true true
true true true

not:

Input Result
false true
true false

Note that the not operator only takes one input. We can use these operators, along with parenthesis, to create the equivalent of mathematical expressions: 

(false or true) = true
(not true) = false
(not false) = true
((not true) and (false or true)) = false
((not false) and (false or true)) = true
In reality, these different operators are implemented as 'logic gates', tiny electronic circuits that behave the same as the different operators, using a powered wire as true, and an unpowered wire as false. These logic gates can be wired together so as to simulate parenthesis.

Advanced Logic Gates

There are, in fact, many more logic gates than just the three I have listed: the xor gate, which is only true when exactly one of it's inputs is true:

Left Right Result
false false false
true false true
false true true
true true false

and the implies gate, which is only true when exactly one of it's inputs is true and one is false:

Left Right Result
false false false
true false true
false true false
true true false

Binary

Computers use a somewhat different number system then people do. When we do math, we use what is called a base-10 number system, because there are 10 different symbols: 0 through 9. The system computers use is a binary2 system, which is where the 'zeros and ones' come from: the two digits in binary, 0 and 1. OK, so what does this binary system look like? To understand this we need to go waaay back to how everyone first learns numbers, the place system. The place system, for those of you who are to far from elementary school goes like this:

Amount 10000 1000 100 10 1 0.1 0.01 0.001
Power 104 103 102 101 100 10-1 10-2 10-3

You can write any number by putting the decimal point between 1 and 0.1, and adding digit1*placevalue1+digit2*placevalue2+digit3*placevalue3... for all the digits. Binary works on a very similar principle, but using powers of 2:

Amount 16 8 4 2 1 0.5 0.25 0.125
Power 24 23 22 21 20 2-1 2-2 2-3

To write a number in binary, it's the same as in decimal, but with 0 and 1. For instance, to write the number five:

Amount 16 8 4 2 1
Power 24 23 22 21 20
Value 0 0 4 0 1
Digit 0 0 1 0 1

This is equal to five because there is a 1 in the fours place, and a one in the ones place, and 4+1=5.
