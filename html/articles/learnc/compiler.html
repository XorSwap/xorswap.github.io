How to install and use GCC on Windows

The exact name of the program we will be installing is MinGW, which stands for MINimal Gnu for Windows. To begin, make a folder called GCC of MinGW or something like that to install it to. To download the installer for the installer, visit http://sourceforge.net/projects/mingw/files/ and hit the link that says "Download mingw-get-setup.exe (86.5 kB)".
