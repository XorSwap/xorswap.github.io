default types:
	char      ~ 8 bit
	short     ~ 16 bit
	int       ~ 16 bit
	long      ~ 32 bit
	long long ~ 64 bit
	void      ~ nothing
constructed types:
	<type> *name ~ pointer to type
	<type> <name>\[<length>\] ~ array:
		\[<length>\] can be repeated
	<return_type> (*<name>)(<args>[,...]) ~ function pointer:
		args is a comma separated list of declarations
functions:
	<return_type> <name>(<args>[,...])
	\{
		<code>
	\}
