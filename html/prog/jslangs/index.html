JS Languages

When I am bored in a class, or anywhere where I have paper and a pen, I like to entertain myself by writing short interpreters for various small programming languages that I know of. I have a basic template that I use to make the interpreters: 
(function(P,_,O)
{
	for(/*code*/; /*code*/; /*code*/)
		/*code*/;
	O(_)
})
P is the program, usually an array of integers, _ is the memory, used to provide an input, and O is a callback function (shorter than return).
